# 1. Apex Scientific

A premium, feature-rich scientific calculator built with Java Swing.

## 🚀 Features
- **Modern Dark UI**: Aesthetic dark theme with gold accents.
- **Scientific Logic**: custom Shunting Yard engine for complex expressions.
- **Constants & Functions**: PI, E, sin, cos, log, ln, sqrt, and more.
- **Integrated History**: real-time sidebar tracking of previous calculations.
- **Memory Management**: M+, MR, MC functions.
- **Responsive Layout**: perfectly centered and fills the frame.
- **Keyboard Support**: fast typing with Enter, Backspace, and Escape.

## 🛠️ Requirements
- Java Development Kit (JDK) 8 or higher.

## 📦 Installation & Running
1. Clone the repository.
2. Compile the source files:
   ```bash
   javac *.java
   ```
3. Run the application:
   ```bash
   java Main
   ```

# 🐍 Snake Game (Java Swing)

## 📌 Description

This is a classic Snake Game developed using Java Swing.
The player controls a snake that moves around the screen, eats food, and grows in size. The game ends if the snake collides with the wall or itself.

---

## 🚀 Features

* 🐍 Smooth snake movement using arrow keys
* 🍎 Random food generation
* ⚡ Dynamic speed increase as score increases
* 💥 Collision detection (wall & self)
* 🎯 Score tracking system
* 🎨 Enhanced UI with snake head and eyes

---

## 🛠️ Technologies Used

* Java
* Swing (GUI)
* AWT (Graphics & Events)

---

## ⚙️ Requirements

* Java JDK 8 or higher
* Any IDE (VS Code / IntelliJ / Eclipse) OR terminal

---
